                                                   PROJECT OVERVIEW
                                                   
This project presents a comprehensive Power Bi data analytics solution for ELECTROHUB, a retail brand spanning multiple products lines(Electronics, Footwear, Clothing, Home Appliances, Acesories, Kitchenware, Bags, and Personal care).

The objective of this project was to transfrom raw, disconnected transactional and dimensional retail data into an interactive, insight-driven dashboard that answers critical business questions regarding sales trends, profitability, customer distributions and promotional performance.
